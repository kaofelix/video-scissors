# Vision

Video Scissors is a fast, focused video editor for quick, streamlined changes.

## Core editing loop

1. open a video
2. scrub precisely
3. apply quick edits such as crop or segment removal
4. reload the current working video
5. continue editing
6. export only when desired

Editing updates the current working video inside the app.
Export is a separate save flow.

## UX direction

The app should feel direct and low-friction.

Key ideas:

- prefer direct manipulation over tool modes
- crop directly on the video surface
- keep timeline editing fast and visually clear
- treat the scrubber/timeline as a signature feature
